# React Weather App

## Live version

[View Live APP](https://alec-react-weather-app.netlify.app)

## Create a file called .env in the root of the folder containing your API key

```
REACT_APP_API_KEY=your_openweathermap_api_key
```

## Install dependencies

```
yarn
or
npm i
```

## Launch the app

```
yarn start
or
npm start
```
